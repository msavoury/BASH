# BASH

## Colors
BASH allows for the usage of colors and font styles in output by means of escape sequences.  These escape general format is:
```
<escape-sequence><font code>
```
### Escape sequences
The following values are valid escape sequences

- ```\e``` 
- ```\033``` 
- ```\x1B``` 


### Codes
**Font Styles**
- ```[0m``` - Normal Text
- ```[1m``` - Bold Text
- ```[2m``` - Dim Text
- ```[4m``` - Underlined Text
- ```[7m``` - Inverted Text
